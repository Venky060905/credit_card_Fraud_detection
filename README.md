# Credit Card Fraud Detection & Risk Analysis

An end-to-end Fraud Detection and Risk Analysis project using Python, SQL, MySQL, Machine Learning, and Tableau to identify fraudulent credit card transactions and visualize risk insights.

## 🚀 Project Overview

This project analyzes credit card transactions to detect fraudulent activities using machine learning and data analytics techniques.

### Key Features
- Data Cleaning & Preprocessing
- Fraud Detection using ML Models
- SQL-based Risk Analysis
- MySQL Database Integration
- Interactive Tableau Dashboard
- Fraud KPI & Transaction Insights
